<a>

L'élément HTML <a> (pour ancre ou anchor en anglais), avec son attribut href, crée un lien hypertexte vers des pages web, des fichiers, des adresses e-mail, des emplacements se trouvant dans la même page, ou tout ce qu'une URL peut adresser. Le contenu de chaque élément <a> doit indiquer la destination du lien. Si l'attribut href est présent, appuyer sur la touche entrée en se concentrant sur l'élément <a> l'activera.

Attributs

Nom Description Valeurs (Exemple)
download Le navigateur traite l'URL lié comme un téléchargement filename
href L'URL vers laquelle pointe l'hyperlien. Les liens ne sont pas limités aux URL basées sur le protocole HTTP - ils peuvent utiliser n'importe quel schéma d'URL pris en charge par les navigateurs. url, mailto, tel
hreflang Indication du langage humain de l'URL lié. Pas de fonctionnalité intégrée. fr
ping Une liste d'URL séparées par des espaces. Lorsque le lien est suivi, le navigateur envoie des requêtes POST avec le corps PING aux URL.
referrerpolicy Exemple no-referrer, no-referrer-when-downgrade, origin, origin-when-cross-origin, same-origin, strict-origin, strict-origin-when-cross-origin (par défaut), unsafe-url
rel La relation de l'URL lié sous forme de types de liens séparés par des espaces.
target L'endroit où afficher l'URL lié, comme le nom d'un contexte de navigation (un onglet, une fenêtre ou une <iframe>). _self, _blank, _parent, _top
type Indique le format de l'URL lié à l'aide d'un type MIME. Pas de fonctionnalité intégrée.

Exemple

Site Internet
<a href="https://example.com"> Site Internet </a>

<abbr>

L'élément HTML abbr représente une abréviation ou un acronyme. Lors de l'inclusion d'une abréviation ou d'un acronyme, il convient de fournir une expansion complète du terme en texte clair lors de la première utilisation, ainsi que l'abréviation pour marquer l'abréviation. L'utilisateur est ainsi informé de la signification de l'abréviation ou de l'acronyme.

Attributs

Cet élément ne prend en charge que les attributs globaux. L'attribut title a une signification sémantique spécifique lorsqu'il est utilisé avec l'élément abbr ; il doit contenir une description complète et lisible par l'homme ou une expansion de l'abréviation. Ce texte est souvent présenté par les navigateurs sous la forme d'une infobulle lorsque le curseur de la souris passe sur l'élément. Chaque élément abbr que vous utilisez est indépendant de tous les autres ; le fait de fournir un titre à l'un d'entre eux n'entraîne pas automatiquement l'ajout du même texte d'expansion aux autres éléments ayant le même contenu textuel.

Exemple

Nous allons voir le HTML qui sert à construire des pages web.

Si l'élément est d'une importance particulière, on utilisera l'élément HTML <strong>.
